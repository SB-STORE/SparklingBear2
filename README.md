# 🚀 React + Vite Project

A fast, modern web application built with **Vite**, **React**, **TypeScript**, **Tailwind CSS**, and **shadcn/ui**.

---

## 🧰 Tech Stack

- ⚡ **Vite** – Next-generation frontend tooling for lightning-fast builds  
- ⚛️ **React** – UI library for building component-based user interfaces  
- 🧩 **TypeScript** – Type-safe JavaScript for fewer bugs and better tooling  
- 🎨 **Tailwind CSS** – Utility-first CSS framework for rapid UI development  
- 🪄 **shadcn/ui** – Beautifully styled, accessible UI components built on Radix UI
